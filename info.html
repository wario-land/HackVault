Welcome to the Wario Land 4 Hack Vault. Here, you can find patch files of Wario Land 4 Hacks.

This website was created by some of the wario-land contributors, using the Bootstrap Framework.

The current Mod Curators are: MarioSpore, Spleen, Kleyman.

If you want your hack to be added, Join our Discord and ask the Mod Curators. A way to submit Hacks via this website will be coming soon. If you find any bugs in the program or this website, contact the developers and web developers, respectively. If you have a suggestion for what can be added, again, ask the developers. If not, the Mod Curators can tell them for you when they get the chance.
